ログ管理プラグイン

Core Protectプラグインは、Minecraftのサーバーでのブロックの変更をログに記録し、元に戻すことができる強力なツールです。以下は、よく使われるコマンドの説明です。

/co inspect (/co i)
◦説明: このコマンドを使用すると、指定したブロックの変更履歴を調査できます。
使用方法: コマンドを入力した後、対象のブロックを右クリックすることで、そのブロックに関する詳細なログ情報を確認できます。
この機能は、特定のプレイヤーが行った変更や破壊を調べるのに役立ちます。

/co near
◦説明: 自分の周りのブロックの変更履歴を表示します。
使用方法: コマンドを入力すると、自分の近くにあるブロックの変更情報をリスト表示します。これにより、特定の範囲内での活動を把握することができます。

Core Protectを活用することで、サーバー内のプレイヤーの行動を監視し、問題が発生した際の迅速な対応が可能になります。

楽しいコマンドの使用方法

Minecraftでは、プレイヤーがさまざまな体勢を取ることができる楽しいコマンドがあります。以下は、そのコマンドの説明です。

/sit
◦説明: プレイヤーが座ることができるコマンドです。周囲の景色を楽しみながら、リラックスした姿勢を取ることができます。

/lay
◦説明: プレイヤーが寝る体制になります。寝転がった姿勢を取り、仲間と一緒にくつろぐことができます。

/crawl
◦説明: プレイヤーが泳ぐ体制になります。水の中での動きがよりリアルに感じられ、泳いでいるように見えます。

/spin
◦説明: プレイヤーが回転します。360度回転することで、周囲の景色を楽しむことができます。

これらのコマンドを使うことで、ゲーム内での体験がより豊かになります。友達と一緒に楽しんでください！

ブロック保護コマンドの使用方法

Minecraftでは、ブロックの保護や操作に役立つコマンドがあります。以下は、そのコマンドの説明です。

/lock
◦説明: ブロックをロックします。特定のプレイヤーのみが操作できるようになります。

/unlock
◦説明: ロックを解除します。誰でも操作できるようになります。

/cprivate
◦説明: 自分だけ、または特定の人だけが使えるようにブロックを保護します。

/cpublic
◦説明: みんなが使えるようにブロックを保護します。

/cpassword <パスワード>
◦説明: パスワードでブロックを保護します。指定したパスワードを入力しないと操作できなくなります。

/cremove
◦説明: ブロックの保護を解除します。

/cinfo
◦説明: ブロックの保護情報を確認します。

/cmodify public
◦説明: 共有化します。破壊は設置者のみ可能です。

/cmodify donation
◦説明: 他の人がアイテムを入れられるが、取り出せません。

/cmodify display
◦説明: 全プレイヤーにアイテムを見せることができます。

/cmodify password <英数字>
◦説明: パスワードを設定します。

/cunlock <パスワード>
◦説明: パスワードを入力して保護を解除します。

/cmodify <ユーザー名>
◦説明: 指定したユーザーと共有します。

/cautoclose on
◦説明: ドアが自動で閉まる機能をオンにします。

/cautoclose off
◦説明: ドアが自動で閉まる機能をオフにします。

/chopper on
◦説明: ホッパーからの搬入出を受け付けるようにします。

/cpersist
◦説明: 複数のブロックを一度に操作します。/cpersist を使った後に操作コマンド（例: /unlock）を実行すると、右クリックで連続して操作できます。再度 /cpersist を入力すると操作を終了します。

これらのコマンドを使用することで、ブロックの管理がスムーズになります。ぜひ試してみてください！

商店街への店の出し方

村人の卵を手に持った状態で、右クリックをしてチャット欄に「Trading - Shop」と表示させます。

村人の卵を右クリック
◦説明: チャット欄に「Trading - Shop」と表示されます。

その後、アイテムの在庫用に使用したいチェストを右クリックします。

チェストを右クリック
◦説明: 在庫用のチェストを設定します。

Shopを展開したい場所へ村人を召喚するために、再度右クリックします。

村人を召喚するために右クリック
◦説明: 設定した場所に村人を召喚します。

商品を村人へ追加したい場合は、まず在庫用のチェストに買ってもらいたいアイテムを入れておきます。

在庫用のチェストにアイテムを入れる
◦説明: 買ってもらいたいアイテムをチェストに収納します。

その後、シフトキーを押しながら村人を右クリックすると、在庫に追加したアイテムが村人の取引リストに並びます。

シフトしながら村人を右クリック
◦説明: 村人に在庫アイテムを追加します。

アイテムの下にある2マスにはバリアブロックが設定されているため、そこへ買うために必要なアイテムを入れてください。

必要なアイテムをバリアブロックの下へ入れる
◦説明: 取引に必要なアイテムを設定します。

これで村人との商取引が設定され、アイテムの売買が可能になります。
